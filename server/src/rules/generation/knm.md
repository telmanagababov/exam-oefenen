# KNM (Kennis van de Nederlandse Maatschappij) - Exam Rules

## Structure
- Multiple Choice (Situational logic)

## Content Requirements
- Focus on: Work & Income, Healthcare, Education, History/Geography, and Values (Equality/Freedom)
- Question Style: "Situatie-vragen." Describe a scenario (e.g., "Je buurman heeft harde muziek aan.") and ask what is the correct/normal behavior in the Netherlands
- Tone: Fact-based and culturally objective
